# test-widget
see it on [github pages](https://romanbykov.github.io/test-widget/)

## Build Setup:

``` bash
# Download repository:
git clone https://github.com/RomanBykov/test-widget

# Go to the app:
cd test-widget

# Install dependencies:
npm install

# Server with hot reload at http://localhost:3000/
npm start

# Output will be at build
npm run build
```
